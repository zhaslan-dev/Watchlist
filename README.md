# 🎬 Watchlist — Telegram-бот для синхронизации медиа-очереди

[](https://www.python.org/downloads/)
[](https://docs.aiogram.dev/)
[](https://fastapi.tiangolo.com/)
[](https://www.postgresql.org/)

**Watchlist** — это асинхронный сервис для коллективного управления очередью кинопросмотра. Бот автоматизирует процесс выбора контента: от поиска через Kinopoisk API до финальной оценки и ведения истории просмотров.

-----

## 🛠 Ключевой функционал

  * **Мультиконтекст:** Корректная обработка логики в **личных сообщениях** (упрощенный аппрув) и **групповых чатах** (система голосования).
  * **Система оценки:** Новый модуль пост-просмотровой оценки (1–10) с сохранением в реляционную базу данных.
  * **Усиленная приватность:** Валидация прав доступа к управлению очередью и защищенная обработка пользовательских данных.
  * **Alert System:** Система мгновенных уведомлений о статусе голосования, достижении кворума и новых добавлениях.
  * **Web-интерфейс:** Внешний сайт для удобного бесплатного просмотра выбранного контента.
  * **Интеграция с VOD:** Автоматическая генерация deep-links для быстрого перехода в онлайн-кинотеатры (Кинопоиск, Иви, Okko).

-----

## 🏗 Технологический стек

  * **Backend:** Python 3.11+, `aiogram 3.x` (Bot API), `FastAPI` (Web/API).
  * **Database:** `PostgreSQL` с асинхронным драйвером `asyncpg`.
  * **ORM:** `SQLAlchemy 2.0` (Declarative Mapping).
  * **Migration:** `Alembic`.
  * **Validation & Settings:** `Pydantic v2`.
  * **Observability:** `Prometheus` (метрики) + `Loguru` (структурированное логирование).
  * **DevOps:** `Docker Compose`, `Poetry` (dependency management).
  * **Testing:** `Pytest` (полное покрытие новой логики чатов и алертов).

-----

## 🏛 Архитектурные принципы

Проект реализован в соответствии с принципами **Clean Architecture** (Луковичная архитектура) и **SOLID**:

1.  **Domain:** Чистые бизнес-сущности и интерфейсы репозиториев.
2.  **Application:** Use-cases (логика голосования, правила принятия фильма).
3.  **Infrastructure:** Реализация внешних зависимостей (API Кинопоиска, DB-адаптеры, Telegram-транспорт).

## 🚀 Быстрый старт

### 1\. Окружение

Склонируйте репозиторий и настройте переменные в `.env`:

```env
BOT_TOKEN=54321:AA...
KINOPOISK_API_KEY=your_key_here
DATABASE_URL=postgresql+asyncpg://user:pass@localhost:5432/watchlist
VOTE_THRESHOLD=2  # Разница голосов для принятия решения
```

### 2\. Развертывание

Рекомендуемый способ запуска через Docker:

```bash
docker-compose up -d --build
```

Для применения миграций:

```bash
docker-compose exec bot alembic upgrade head
```

## 📖 Справочник команд

| Команда | Описание |
| :--- | :--- |
| `/search` | Инициация поиска фильма через API с пагинацией |

| `/settings` | Конфигурация приватности и уведомлений |
